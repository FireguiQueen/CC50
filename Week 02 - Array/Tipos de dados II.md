# Tipos de dados 
Cada tipo de dado ocupa uma quantidade fixa de espaço, e na verdade, diferentes sistemas de computador variam na quantidade de espaço realmente usado para cada tipo. Mas em geral, será desta forma:
- `bool`    _1 byte_
- `char`    _1 byte_
- `double`  _8 bytes_
- `float`   _4 bytes_
- `int`     _4 bytes_
- `long`    _8 bytes_
- ..

</br>

# RAM (random acess memory) & HD (hard disk) 
Há muita confusão no que se diz a respeito de memória RAM e HD. O HD (hard disk) é um hardware utilizado para armazenar dados, ele não apenas os salva, mas também pode fazer a leitura do mesmo.
